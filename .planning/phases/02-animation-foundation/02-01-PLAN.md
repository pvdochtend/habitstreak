<task type="auto">
  <name>Task 2: Add glassmorphism CSS utilities</name>
  <files>src/app/globals.css</files>
  <action>
Add glassmorphism utility classes to globals.css in a new section:

```css
/* ============================================
   GLASSMORPHISM UTILITIES
   ============================================ */
.glass {
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  background: hsl(var(--card) / 0.7);
  border: 1px solid hsl(var(--border) / 0.3);
}

.glass-subtle {
  backdrop-filter: blur(8px);
  -webkit-backdrop-filter: blur(8px);
  background: hsl(var(--card) / 0.5);
  border: 1px solid hsl(var(--border) / 0.2);
}

.glass-strong {
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  background: hsl(var(--card) / 0.85);
  border: 1px solid hsl(var(--border) / 0.4);
}

/* Fallback for browsers without backdrop-filter */
@supports not (backdrop-filter: blur(12px)) {
  .glass,
  .glass-subtle,
  .glass-strong {
    background: hsl(var(--card) / 0.95);
  }
}
```

Place this after the theme definitions but before the animation keyframes section.
  </action>
  <verify>Open browser DevTools, inspect element with .glass class, see backdrop-filter applied</verify>
  <done>Three glass utility classes available: .glass, .glass-subtle, .glass-strong with proper fallbacks</done>
</task>
